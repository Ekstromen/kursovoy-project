											Вас приветствует головоломка 'Нурикабе-линия' !

	На экране у вас появиться игровое поле на котором будут символы '*', которые обозначают сушу, а также на игровом поле будут цифры, которые обозначают площадь острова включая клетку с самой цифрой.
Задача игрока: поставить клетки моря '#' так, чтобы острова соответствовали площади прилежащих к ним цифр, при этом соблюдая правила головоломки.
Правила: пять клеток '#' не должны стоять в ряд; область размером 2 x 2 клетки может состоять из '#'.

	Пример решеной головоломки:
								# * * # * 3 # #
								* # * # * # 6 *
								3 # * # # # * *
								* # 5 # # # * *
								# # # * * 3 # #
								# * # # # # * 6
								# 2 # # # # * *
								# # 2 * # # * *

	Для перемещения используй клавиши 'wasd', для замены символов используй клавишу 'f', для завершения игры 'e' (Символы должны быть строчные и на английской раскладке).
Нажмите любую клавишу чтобы продолжить...
